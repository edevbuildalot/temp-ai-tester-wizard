# Temp AI Tester Wizard

Paste code. Get tests. Ship faster.

A single-page web app that converts code snippets into real, runnable test cases using Gemini AI. No signup, no configuration, instant results.

## Getting Started

```bash
# Install dependencies
npm install

# Set up your Gemini API key
cp .env.example .env.local
# Edit .env.local and add your GEMINI_API_KEY

# Start dev server
npm run dev
```

## Build

```bash
npm run build
npm start
```
